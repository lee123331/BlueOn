BlueOn

결제 · 환불 처리 안내

BlueOn은 고객과 전문가가 신뢰할 수 있는 거래를 할 수 있도록 에스크로 기반 보호 시스템과 명확한 단계별 환불 기준을 운영합니다.

1. 결제 방식 및 보호 시스템

✔ 모든 결제는 PG사를 통해 안전하게 처리됩니다.

✔ 결제 금액은 에스크로 계좌에 보관되며, 고객 승인 시에만 전문가에게 지급됩니다.

✔ 작업 완료 전에는 대금이 자동으로 정산되지 않습니다.

✔ BlueOn은 중립적인 기준으로 분쟁을 검토하여 최종 판단합니다.

2. 환불 가능 조건

📌 작업 시작 전 환불 요청
→ 결제 금액의 100% 환불


📌 1차 시안 제출 후 환불 요청
→ 결제 금액의 50% 환불 가능
(초안 작업물이 제공된 상태)


📌 수정 작업 진행 중 환불 요청
→ 환불 불가
(고객의 요청에 따라 2차 작업이 진행된 경우)


📌 최종본 제공 이후
→ 환불 불가

3. 진행 단계별 상세 환불 기준
A단계 — 작업 시작 전
✔ 파일, 시안 등 어떠한 작업물도 전달되지 않은 상태
→ 100% 환불 가능
B단계 — 1차 시안 제출
✔ 로고 초안, 상세페이지 기본 틀 등 기본 산출물이 전달된 단계
→ 50% 환불 가능
C단계 — 수정 작업 진행 중
✔ 고객이 보완 요청을 하여 후속 작업이 진행되는 단계
→ 환불 불가
D단계 — 최종 작업 직전
✔ 전체 작업의 대부분이 완료된 상태
→ 환불 불가
E단계 — 최종 결과물 제공 완료
✔ 고객이 최종 파일을 확인하거나 다운로드한 상태
→ 환불 불가
4. 환불이 가능한 구체적 사례
✔ 작업 시작 전 취소 요청 → 100% 환불
✔ 1차 시안이 마음에 들지 않아 환불 요청 → 50% 환불
✔ 전문가가 작업 불가 사유를 밝힌 경우 → 100% 환불
5. 환불이 어려운 사례
✘ 수정 요청 후 작업이 이미 진행된 경우
✘ 작업 방식 또는 스타일 차이로 인한 단순 변심
✘ 최종본 전달 이후 환불 요청
⚠ 환불 분쟁이 발생한 경우,
BlueOn 운영팀은 작업 기록 · 메시지 내역 · 제출 산출물을 검토하여 중립적이고 공정한 기준으로 최종 환불 여부를 판단합니다.

전문가가 ‘거부하기’를 선택한 경우에도,
최종 환불 여부는 BlueOn 운영팀이 결정합니다.
