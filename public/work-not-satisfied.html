BlueOn

작업이 마음에 들지 않을 때

작업물이 기대와 다를 수 있습니다.
BlueOn은 고객과 전문가가 합리적으로 조정할 수 있도록 명확한 수정·조정 절차를 제공합니다.

1. 먼저, 수정 요청이 가능합니다
✔ 모든 서비스에는 기본적으로 정해진 수정 횟수가 포함되어 있습니다.
✔ 수정 범위는 서비스 등록 시 기재된 내용(기능/디자인/문구 등)을 기준으로 합니다.
✔ 전문가에게 구체적인 피드백을 제공하면 수정이 빠르게 진행됩니다.
2. 수정이 불가능한 경우
✘ 처음 합의한 범위를 벗어난 새로운 기능·디자인 요구
✘ 작업물이 이미 확정된 상태에서의 방향 전면 변경
✘ 전문가가 완성한 파일을 이미 다운로드·사용한 경우
3. 그래도 조정이 필요한 경우
✔ 전문가와 협의하여 부분 수정 또는 추가 비용으로 확장 가능
✔ 조정이 어려운 경우 BlueOn이 중립적 판단으로 중재를 진행합니다.
4. 완성 이후 환불은 불가능합니다
✔ 작업물이 최종 전달된 시점부터는 환불이 불가합니다.
✔ 단, 전문가의 명백한 작업 불량이 확인될 경우에만 예외적으로 처리됩니다.
⚠ 분쟁 발생 시 BlueOn 운영팀은 제출된 작업 파일, 대화 내역, 계약 조건을 확인하여 공정하고 객관적인 기준으로 판단합니다.
